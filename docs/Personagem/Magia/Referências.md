## Arcana
| Complexidade (Custo de Mana) | Caminhos         | Exemplos de Efeito                                                                                                                                             |
| ---------------------------- | ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Baixa Complexidade (1-3)     | **Meta**         | Infundir partes do corpo, acelerar tempos de reação, criar uma aura protetora fraca.                                                                           |
| Média Complexidade (4-6)     |                  | Impregnar uma arma com energia mágica, infundir itens, ganhar força temporária, velocidade ou saúde.                                                           |
| Alta Complexidade (7-9)      |                  | Transmutar um objeto, ou infundi-lo com poderes mágicos permanentemente, remodelar o tecido da realidade por um breve momento.                                 |
| ---                          | **Elemental**    |                                                                                                                                                                |
| Baixa Complexidade (1-2)     |                  | Conjurar brilhos, um lampejo de chama ou uma brisa suave. Criar uma fonte de água, extinguir um pequeno fogo, atirar pedras.                                   |
| Média Complexidade (3-6)     |                  | Invocar um redemoinho controlado em uma área localizada, atirar pequenas bolas de fogo, erguer pedregulhos do chão, controlar pequenos corpos de água.         |
| Alta Complexidade (7-9)      |                  | Desencadear poderosas explosões de chamas, bolas de fogo, conjurar grandes turbilhões, erguer e dobrar a terra.                                                |
| ---                          | **Luz**          |                                                                                                                                                                |
| Baixa Complexidade (4-5)     |                  | Conjurar pequenas luzes, criar sombras, espelhar ou alterar pequenos objetos através de ilusão óptica.                                                         |
| Média Complexidade (6-8)     |                  | Conjurar espectros, hologramas de luz, transformar a aparência de um espaço pequeno, espelhar perfeitamente a imagem de alguém.                                |
| Alta Complexidade (9-10)     |                  | Distorcer a a aparência de todas as coisas ao redor de uma área considerável, conjurar clones holográficos perfeitos.                                          |
| ---                          | **Espaço/Tempo** |                                                                                                                                                                |
| Baixa Complexidade (4-5)     |                  | Teleportar instantaneamente uma curta distância, retroceder alguns segundos no passado.                                                                        |
| Média Complexidade (6-8)     |                  | Criar um portal entre curtas distâncias, retroceder alguns segundos no passado ou obter um vislumbre do futuro imediato.                                       |
| Alta Complexidade (9-10)     |                  | Desacelerar o tempo em uma área localizada, teleportar instantaneamente para qualquer lugar dentro de um grande raio, criar um portal entre longas distâncias. |

| Alta Complexidade (8-10)     |                | Conjure espíritos elementares, entidades da natureza, incorpore seus poderes.                                        |
